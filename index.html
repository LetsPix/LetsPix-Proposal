Application Overview
Technology Overview
The LetsPix application will allow users to access a library of movies from a variety of media service providers. Movies and TV shows from subscription services such as Netflix, Amazon Prime Video, and Hulu will be discoverable through a variety of methods. The main goal of this application is to provide users a way to examine a variety of movies and TV shows while quickly accessing where one can watch said media from a variety of genres and media providers. The LetsPix application will employ a few key technologies. Most importantly, we will be using HTML and Bootstrap (CSS) to present movies to users on a static client oriented website. We will also be utilizing a MongoDB Atlas database to hold movie information. We will also be utilizing an API format for retrieving movie information from the database to avoid loading excess information onto user devices. This movie information will be formatted using JSON. To facilitate having different client and server environments, we will also be employing node and express.
